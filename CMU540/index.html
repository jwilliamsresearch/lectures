Assessment Overview

CMU540: Cyber Security has a comprehensive assessment structure designed to evaluate practical security skills and theoretical understanding.

Security Assessment & Penetration Testing Report (100%)

Project Type: Individual security assessment and penetration testing

Scope: Web application or network security assessment

Requirements: Comprehensive security analysis and ethical hacking report

Due: End of module (consultation with tutor required)

Project Requirements

  • Security Assessment: Comprehensive vulnerability assessment of target system
  • Penetration Testing: Ethical hacking demonstration with proper documentation
  • Risk Analysis: Detailed risk assessment and mitigation strategies
  • Technical Report: Professional security assessment report
  • Ethical Considerations: Legal and ethical compliance documentation
  • Presentation: Security findings presentation to stakeholders

Learning Outcomes Assessed

  • Analyse and evaluate cyber security threats and vulnerabilities
  • Apply security assessment methodologies and penetration testing techniques
  • Design and implement security controls and countermeasures
  • Conduct digital forensics and incident response procedures
  • Develop security policies and compliance frameworks
  • Demonstrate ethical hacking practices and professional conduct
