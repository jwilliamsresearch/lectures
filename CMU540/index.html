CMU540

Cyber Security

Network & Web Application Security

Birmingham Newman University | 20 Credits | Undergraduate

Module Overview

Explore network and web application security in our OS³ Cyber Security Lab. Learn threat analysis, defensive strategies, and ethical hacking fundamentals with hands-on practice.

Lecture Slides

  • 1.Cyber Security Fundamentals & Lab Setup — Introduction and CIA triad
  • 2.WEB-ACCESS-01 - Broken Access Control — OWASP Top 10 #1
  • 3.WEB-CRYPTO-01 - Cryptographic Failures — OWASP Top 10 #2
  • 4.WEB-SQL-01 - SQL Injection — OWASP Top 10 #3
  • 5.WEB-XSS-01 - Cross-Site Scripting — Client-side attacks
  • 6.WEB-CSRF-01 & WEB-CONFIG-01 — CSRF and Misconfiguration
  • 7.WEB-VULN-01 & WEB-AUTH-01 — Vulnerable Components & Authentication
  • 8.WEB-DATA-01 & WEB-LOG-01 — Data Exposure & Logging
  • 9.WEB-SSRF-01 & WEB-UPLOAD-01 — SSRF & File Upload Security
  • 10.HACKATHON — Practical security challenge
  • 11.NET-PROTO-01, NET-SCAN-01 & NET-DNS-01 — Network Security Fundamentals
  • 12.NET-TRAFFIC-01, NET-FIREWALL-01 & Final Assessment — Network defense and review
