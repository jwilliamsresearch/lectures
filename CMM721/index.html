Web Application Development

Full-Stack Development with Python & Django

Birmingham Newman University | 20 Credits | Masters Conversion

Module Overview

Intensive masters-level introduction to full-stack development. Master HTML5, CSS3, JavaScript, Python, Django, and React through comprehensive hands-on projects.

Lecture Slides

  • 1.Module Introduction & Web Architecture — Full-stack overview
  • 2.HTML5 & Semantic Structure — Modern markup
  • 3.CSS3 & Responsive Design — Modern styling
  • 4.JavaScript Fundamentals — Core concepts and DOM
  • 5.Asynchronous JavaScript & APIs — Promises and fetch
  • 6.Django Basics — Framework introduction
  • 7.Django ORM & CRUD — Database operations
  • 8.Django REST Framework — API development
  • 9.React & State Management — Component-based UI
  • 10.Web App Critique — Analysis methods
  • 11.Comparative Critique — Evaluation techniques
  • 12.Project Finalisation — Review and deployment
