Fundamentals of Web Design

HTML5, CSS3 & JavaScript

Birmingham Newman University | 20 Credits | Undergraduate

Module Overview

Master the essentials of modern web design with HTML5, CSS3, and JavaScript. Build responsive, accessible websites for real clients while learning industry-standard practices.

Lecture Slides

  • 1.Introduction to HTML — Web fundamentals and HTML basics
  • 2.HTML Forms and Input — Semantic HTML and accessibility
  • 3.CSS Fundamentals — Selectors and properties
  • 4.CSS Layout and Box Model — Flexbox and Grid
  • 5.CSS Flexbox and Grid — Media queries and mobile-first
  • 6.JavaScript Basics — Variables and functions
  • 7.JavaScript Events and Interactivity — Event handling
  • 8.JavaScript Advanced Concepts — Client-side validation
  • 9.Web APIs and External Libraries — Smooth interactions
  • 10.Project Planning and Development — Optimization techniques
  • 11.Advanced Topics and Future Trends — WCAG guidelines
  • 12.Final Project and Assessment — Professional workflow
